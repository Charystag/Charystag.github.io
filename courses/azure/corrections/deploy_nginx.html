1. Personnalisation de la Page Web

Commande pour modifier index.html :

sudo bash -c 'echo "<html>
<head>
<title>Page Personnalisée</title>
<style>body { font-family: Arial; text-align: center; margin-top: 50px; }</style>
</head>
<body>
<h1>Bienvenue sur mon serveur Nginx !</h1>
<p>Date actuelle : $(date)</p>
</body>
</html>" > /var/www/html/index.html'

2. Installation et Configuration d'Apache

Installez Apache et configurez le port 8080 :

sudo apt install apache2 -y
sudo sed -i 's/Listen 80/Listen 8080/' /etc/apache2/ports.conf
sudo systemctl restart apache2

Créez une page HTML personnalisée pour Apache :

sudo bash -c 'echo "<html><body><h1>Bienvenue sur Apache !</h1><p>Port 8080</p></body></html>" > /var/www/html/index_apache.html'

3. Automatisation avec un Script Bash

Exemple de script Bash :

#!/bin/bash
# Mise à jour
sudo apt update -y

# Installation de Nginx
sudo apt install nginx -y

# Configuration de la page
sudo bash -c 'echo "<html><body><h1>Page déployée via script Bash</h1></body></html>" > /var/www/html/index.html'

# Démarrage de Nginx
sudo systemctl restart nginx
sudo systemctl enable nginx

echo "Nginx installé avec succès !"

Exécutez le script :

chmod +x deploy_nginx.sh
./deploy_nginx.sh

4. Nettoyage des Ressources

Supprimez le groupe de ressources :

az group delete --name GroupeVMWeb --yes --no-wait

Conclusion

Grâce à cet exercice, vous avez appris à :

  • Créer et configurer une Machine Virtuelle Azure.
  • Installer et configurer des serveurs web (Nginx et Apache).
  • Automatiser des tâches d’installation avec un script Bash.

Ces compétences vous seront utiles pour des déploiements plus avancés sur Azure, comme l'hébergement d'applications complètes ou l'automatisation des déploiements via des outils DevOps. 🚀
