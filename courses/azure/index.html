Lien vers le cours précédent

Vous retrouverez sur notion le cours précédent. Celui-ci inclus également des présentations de Data-Lake et AWS, qui n'ont pas été couverts dans ce cours qui s'est concentré essentiellement sur Azure. Certaines notions y sont plus développées que dans ce cours-ci tandis que d'autres notions vu pendant ce cours n'y sont pas abordées, voyez donc cette suite de document comme un complément aux cours que vous avez suivi.

Séries de Vidéos Azure

Pour ceux qui préfèrent l'apprentissage vidéo et qui souhaitent retrouver et approfondir les notions abordées en cours sur ce format, voici deux chaînes youtubes qui vous permettront d'approfondir votre compréhension de Microsoft Azure

Adam Marczak

Adam est un MVP (Most Valuable Professional, soit un expert dont la valeur en tant qu'enseignant à été reconnue par Microsoft), dont je me suis largement inspiré pour faire ce cours tant par la structure que par les graphiques utilisés. Vous pouvez consulter sa playlist sa chaîne sur youtube.

Cloud Monk

Cloud Monk illustre les concepts du cloud avec des dessins simples et des histoires concrètes pour vous permettre de comprendre les cas d'application et les enjeux des solutions cloud que vous designez. Vous pouvez retrouver ses vidéos sur sa chaîne

Documentation Microsoft

Vous pourrez retrouver toutes les notions abordées en cours dans la documentation officielle de Azure. Pour chaque produit, celle-ci inclus :

  • Des Explications des Concepts
  • Des Guides Pratiques
  • Des Guides de démarrage rapide
  • Des Guides de Bonnes Pratiques

De plus, pour les produits pour lesquels cela est pertinent, la documentation inclus des tutoriels pour détailler des cas d'application pratique de ces différents produits
