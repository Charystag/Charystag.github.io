Exercice : Hébergement d'un Site Web Statique avec Azure Blob Storage

Objectif

Apprendre à :

  1. Configurer un conteneur Blob Storage pour héberger un site web statique.
  2. Uploader des fichiers HTML et CSS.
  3. Accéder au site web via l’URL publique.

Étapes

Étape 1 : Créer un Compte de Stockage

  1. Connectez-vous au Portail Azure :
    https://portal.azure.com.

  2. Créez un compte de stockage :

    • Allez dans Créer une ressource > Stockage > Compte de stockage.
    • Remplissez les champs :
      • Nom : Par exemple mystaticwebstorage.
      • Région : westeurope.
      • Performance : Standard.
      • Redondance : Locally-redundant storage (LRS).
    • Cliquez sur Revoir + Créer, puis sur Créer.

Étape 2 : Configurer le Compte de Stockage pour l’Hébergement Statique

  1. Une fois le compte de stockage créé, accédez-y dans le portail Azure.

  2. Allez dans la section Données et stockage > Page Web statique.

  3. Activez l’hébergement statique :

    • État : Active.
    • Document d’index : index.html.
    • Document d’erreur (facultatif) : error.html.
    • Cliquez sur Enregistrer.

  4. Notez l’URL du point de terminaison (exemple : https://mystaticwebstorage.z13.web.core.windows.net).

Étape 3 : Créer les Fichiers HTML et CSS

  1. Préparez les fichiers locaux :

    • Fichier index.html :

      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Static Website</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Bienvenue sur mon site web statique</h1>
    <p>Ce site est hébergé avec Azure Blob Storage.</p>
</body>
</html>

    • Fichier styles.css :

      body {
    font-family: Arial, sans-serif;
    text-align: center;
    margin-top: 50px;
}

h1 {
    color: #4CAF50;
}

p {
    font-size: 18px;
}

  2. Enregistrez ces fichiers dans un répertoire local, par exemple : /chemin/vers/mon-site.

Étape 4 : Uploader les Fichiers dans Blob Storage

Option 1 : Utiliser le Portail Azure

  1. Accédez à la section Conteneurs du compte de stockage.

  2. Ouvrez le conteneur par défaut créé pour l’hébergement statique ($web).

  3. Cliquez sur Télécharger et uploadez les fichiers index.html et styles.css.

Option 2 : Utiliser Azure CLI

  1. Assurez-vous que vous êtes connecté à Azure CLI :

    az login

  2. Uploader les fichiers dans le conteneur $web :

    az storage blob upload-batch \
  --account-name mystaticwebstorage \
  --destination '$web' \
  --source /chemin/vers/mon-site

Étape 5 : Accéder au Site Web

  1. Ouvrez l’URL du point de terminaison dans un navigateur :

    https://mystaticwebstorage.z13.web.core.windows.net

  2. Vous devriez voir votre page HTML s’afficher avec le style CSS appliqué.

Entraînement (Extension non-guidée)

Tâches à Réaliser

  1. Ajoutez d’autres pages :

    • Créez un fichier about.html avec des informations supplémentaires et mettez-le en lien depuis index.html.

  2. Personnalisez le site :

    • Ajoutez des images et des scripts JavaScript pour dynamiser le site.

  3. Gérez les erreurs :

    • Créez un fichier error.html qui sera affiché pour les pages non trouvées (404).

  4. Automatisez l’upload des fichiers :

    • Écrivez un script Bash ou PowerShell pour automatiser le processus de mise à jour des fichiers.
