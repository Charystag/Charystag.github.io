Full Stack Application Setup: Frontend and Backend Connection

Introduction

This guide provides a structured setup for a full-stack application, covering the backend API and frontend integration. The frontend is built with React + TypeScript, using Axios for API requests and React Router for navigation. The backend is powered by Node.js, Express, and MongoDB.

üöÄ Backend Setup (Recap)

Before setting up the frontend, make sure the backend is up and running. The backend exposes a RESTful API that the frontend will interact with.

  • The backend is a Node.js Express server, connected to MongoDB via Mongoose.
  • It exposes the following API endpoints:
MethodEndpointDescription
GET/api/productsRetrieve all products
GET/api/products/:idRetrieve a single product
POST/api/productsCreate a new product
PUT/api/products/:idUpdate a product
DELETE/api/products/:idDelete a product

üîó API Base URL: http://localhost:5100/api/products

Ensure the backend is running before moving on to the frontend.

2 Creating the Routing System

We'll set up React Router in App.tsx.

App.tsx

import { BrowserRouter, Routes, Route } from 'react-router-dom';
import Home from './components/Home';
import Create from './components/Create';
import Read from './components/Read';
import 'bootstrap/dist/css/bootstrap.min.css';

function App() {
  return (
    <BrowserRouter>
      <Routes>
        <Route path='/' element={<Home />} />
        <Route path='/create' element={<Create />} />
        <Route path='/read/:id' element={<Read />} />
      </Routes>
    </BrowserRouter>
  );
}

export default App;

Now we have three pages:

  • Home (/) ‚Äì Lists all products
  • Create (/create) ‚Äì Form to add a new product
  • Read (/read/:id) ‚Äì View a single product

import { Product } from '../types.ts';

interface FormField {
  name: string;
  type: "text" | "number";
  placeholder: string;
}

const createFormField = (
  field: FormField,
  values: Product,
  setValues: any
) => {
  return (
    <div className="form-group" key={field.name}>
      <label htmlFor={field.name}>
        {field.name}
      </label>
      <input
        type={field.type}
        name={field.name}
        placeholder={field.placeholder}
        className="form-control"
        value={values[field.name]}
        onChange={e => {
          setValues(prev => ({
            ...prev,
            [field.name]: field.type === "number" ? Number(e.target.value) || 0 : e.target.value
          }));
        }}
      />
    </div>
  );
};

const createForm = (
  fields: FormField[],
  values: Product,
  setValues: any,
  handleSubmit: any
) => {
  return (
    <form onSubmit={handleSubmit}>
      {fields.map(field => createFormField(field, values, setValues))}
      <button className="btn btn-success">Submit</button>
    </form>
  );
};

5Ô∏è‚É£ Viewing a Single Product

üìå Read.tsx

import { useParams, Link } from 'react-router-dom';
import { useEffect, useState } from 'react';
import axios from 'axios';

interface Product {
  _id: string;
  name: string;
  price: number;
  image: string;
}

function Read() {
  const { id } = useParams<{ id: string }>();
  const [product, setProduct] = useState<Product | null>(null);

  useEffect(() => {
    const fetchProduct = async () => {
      try {
        const res = await axios.get(`http://localhost:5100/api/products/${id}`);
        setProduct(res.data);
      } catch (error) {
        console.error(error);
      }
    };
    fetchProduct();
  }, [id]);

  if (!product) return <div>Loading...</div>;

  return (
    <div className="container">
      <h1>{product.name}</h1>
      <p>Price: ${product.price}</p>
      <img src={product.image} alt={product.name} width="200" />
      <br />
      <Link to="/" className="btn btn-primary">Back to Home</Link>
    </div>
  );
}

export default Read;
