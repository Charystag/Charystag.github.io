Working with Lists in Python

Objective:

Learn to create, manipulate, and iterate over lists to store and process collections of data.

Description of the Challenge:

  1. Create a List:

    • Create a list containing the names of your five favorite fruits.

  2. Access Specific Items:

    • Print the first and last fruit in the list using indexing.

  3. Modify the List:

    • Add another fruit to the list.
    • Print the updated list to verify the addition.

  4. Iterate Over the List:

    • Use a loop to print each fruit in the list, one by one.

  5. Optional Extension:

    • Sort the list alphabetically and print the sorted list.
Hints
  • Lists in Python use zero-based indexing. Access the first element with list[0] and the last element with list[-1].
  • Use the append() method to add an item to the list.
  • A for loop is ideal for iterating over list items.
  • To sort the list, use the sort() method.

Refer to the Python Lists documentation for more guidance.
