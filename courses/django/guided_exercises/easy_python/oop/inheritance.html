Reusing Code with Inheritance in Python

Objective:

Understand how to extend functionality using inheritance and override methods in a subclass.

Description of the Challenge:

  1. Create a Puppy Subclass:

    • Define a new class Puppy that inherits from the Dog class.

  2. Add a New Attribute:

    • Add an additional attribute training_level to the Puppy class. Initialize it through the constructor (__init__).

  3. Override the bark() Method:

    • Redefine the bark() method in the Puppy class to print: 
      "Yip! I'm just a puppy."

  4. Test the Classes:

    • Create instances of both Dog and Puppy.
    • Call the bark() method on both objects to observe the overridden behavior.
    • Access the training_level attribute on the Puppy object to confirm it’s correctly set.

  5. Optional Extension:

    • Add a method train() to Puppy that increases the training_level by 1 and prints the updated level.
Hints

  • Use the super() function to call the parent class constructor and initialize shared attributes.

    def __init__(self, name, breed, age, training_level):
    super().__init__(name, breed, age)
    self.training_level = training_level

  • Override the bark() method by redefining it in the Puppy class.

  • Inherit all the parent class’s behavior unless specifically overridden.

Refer to the Python Inheritance documentation for more information.
