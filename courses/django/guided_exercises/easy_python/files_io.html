Basic File Handling in Python

Objective:

Learn how to perform fundamental file operations such as creating, writing to, and reading from text files.

Description of the Challenge:

  1. Write to a File:

    • Create a script that:
      • Opens a file named greetings.txt in write mode.
      • Writes the string "Hello, world!" to the file.
      • Closes the file (or use a with statement to handle it automatically).

  2. Read from the File:

    • Create another script that:
      • Opens the greetings.txt file in read mode.
      • Reads its contents.
      • Prints the content to the console.

  3. Optional Extension:

    • Append a new line with another greeting (e.g., "Hi there!") to the file and print the updated file content.
Hints
  • Use the open() function with mode 'w' for writing and 'r' for reading.
  • The with statement is recommended for safe file handling (e.g., with open('file.txt', 'w') as file:).
  • Use the write() method to add text to a file.
  • Use the read() method to retrieve file content.

Refer to the Python File Handling documentation for further guidance.
