Example 1: Classical Case (Beginner, Conceptual, Python)

Input:

{
  "question": "Which of the following symbols is used for comments in Python?",
  "rules": {
    "topic": "Python",
    "programming_language": "Python",
    "difficulty": "easy",
    "type": "multiple_choice",
    "purpose": "conceptual",
    "background": "No prior programming experience."
  }
}

Expected Output:

{
  "answers": {
    "options": [
      {"text": "#", "correct": true},
      {"text": "//", "correct": false},
      {"text": "/*", "correct": false},
      {"text": "--", "correct": false}
    ],
    "explanation": "In Python, comments start with the '#' symbol. Other symbols like '//' or '/*' are used for comments in different languages.",
    "documentation_links": [
      {
        "url": "https://docs.python.org/3/tutorial/introduction.html#comments",
        "description": "Python Official Documentation: Comments"
      }
    ]
  }
}

Rationale:

  • Straightforward question, simple conceptual difficulty, and a suitable link is provided.

Rationale:

  • A problem-solving question at medium difficulty. The explanation walks through the code logic. Only one link is given since range() is directly relevant.

      {"text": "False", "correct": false}
    ],
