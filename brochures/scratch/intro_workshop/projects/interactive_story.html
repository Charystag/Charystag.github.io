Stage d'introduction à Scratch

Projet 3 : Créer une Histoire Interactive

Description

Dans ce projet, les enfants donneront vie à une histoire interactive en utilisant :

  • Plusieurs personnages pour représenter les acteurs de leur histoire.
  • Différentes scènes pour illustrer les étapes ou les lieux clés de l’histoire.
  • Interactions dynamiques entre les personnages et les éléments de l’histoire.

Notions Apprises

Les participants découvriront :

  • Blocs de Sons : Ajouter des effets sonores et des dialogues pour enrichir l'expérience narrative.
  • Blocs de Scènes : Changer de scène pour avancer dans l’histoire ou marquer une transition.
  • Blocs de Communication : Utiliser des messages pour synchroniser les actions entre personnages et objets.

Objectifs Pédagogiques

  • Narration Créative : Développer des compétences en storytelling et exprimer des idées à travers le code.
  • Coordination : Apprendre à synchroniser les actions et dialogues entre différents objets pour une histoire cohérente.
  • Débogage : Identifier et résoudre des problèmes comme :
    • Des actions mal synchronisées entre les personnages.
    • Des transitions incorrectes entre les scènes.

Durée estimée : 2 à 5 heures

Le temps peut varier selon le nombre de scènes et de personnages, ainsi que le niveau de personnalisation souhaité.
